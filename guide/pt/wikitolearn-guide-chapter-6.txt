Os modelos são usados para adicionar notas recorrentes nas páginas de forma consistente, para criar áreas de navegação e para oferece funcionalidades avançadas, como a portabilidade abrangente do texto.

== Inserir modelos ==

Para inserir modelos, poderá usar a função <code>Inserir->Modelo</code> do [[Manual/The Visual Editor|Editor Visual]].
Escreva o nome do modelo que deseja inserir e seleccione o correcto.

[[File:Theorem_insertion_wtl.gif]]

=== Usar os parâmetros dos modelos ===

Alguns modelos poderão ter parâmetros com nomes.
Poderá preencher um desses campos se carregar no nome do campo e indicar o valor desejado.

== Inserir ambientes ==
Para a matemática, normalmente precisa de marcar as coisas como sendo um Teorema, uma Prova, uma Definição ou outra entidade.
Podê-lo-á fazer se carregar no botão "Adicionar um ambiente" no editor e escolher o ambiente desejado.

[[File:Theorem_title_wtl.gif]]

Nota: quando adicionar um ambiente, irá ver algum texto chamado "END ENVIRONMENT" (fim do ambiente) ou algo do género. Isto é apenas para sua conveniência e para saber que está a escrever dentro do ambiente seleccionado, mas não irá aparecer na página apresentada ao utilizador.
