== Código ==
Todo o nosso código é aberto e podê-lo-á encontrar no '''[https://github.com/WikiToLearn GitHub do WikiToLearn]'''.

===Git===
O Git é uma ferramenta de 'software' usada para gerir projectos grandes de forma colaborativa. O WikiToLearn usa o Git para gerir as actualizações e todas as modificações do projecto.

Primeiro que tudo, terá de '''[https://git-scm.com/ instalar o Git]'''. Se não estiver familiarizado sobre o uso do mesmo, poderá começar com este '''[https://try.github.io/levels/1/challenges/1 tutorial de 15 minutos]'''. É fornecido um tutorial completo do Git pela '''[https://www.atlassian.com/git/ Atlassian]''' ou poderá também tentar o '''[https://www.codecademy.com/learn/learn-git guia da Codeacademy]''' (extremamente simples).

Também lhe sugerimos esta '''[http://ktown.kde.org/~zrusin/git/git-cheat-sheet.svg cábula]''' com os comandos mais usados.

===Docker===
O Docker é uma aplicação usada pelo WikiToLearn para dividir o projecto em diferentes módulos. Para executar o WikiToLearn é necessário '''[https://docs.docker.com/engine/getstarted/ instalar e aprender o Docker]'''.

=== Linguagens de programação ===
Devido à complexidade do nosso projecto, o WikiToLearn aplica diversas tecnologias Web e ambientes de desenvolvimento, mas a maior parte está desenvolvida em: Python, PHP, CSS, HTML e JavaScript.
