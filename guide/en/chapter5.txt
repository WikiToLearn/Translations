VisualEditor is the main tool you can use to create and edit pages on WikiToLearn. Using VisualEditor is very easy and if you already have some experience in any kind of text editing software you will have no problem understanding how VisualEditor works.

== Basic Usage ==

=== Activating VisualEditor ===
When creating a new page, VisualEditor should be enabled by default: the VisualEditor toolbar should appear at the top of the page.

If you want to edit an existing page, just press the <code>Edit</code> button at the top of the page and watch VisualEditor load: after a few seconds it should be ready to use.

[[File:Manual_Visual_editor_newpage.gif]]

=== Editing ===
VisualEditor has more or less the same features and functions of any other word processor software, like "style text" and "structure".

=== Advanced editing ===

==== Insert media ====
To insert media in the text,  just click the <code>Insert</code> option in the VisualEditor toolbar and select <code>Media</code>: you can either upload new files using the <code>Upload</code> tab or <code>Search</code> for existing media both on WikiToLearn or Wikimedia Commons.

[[File:Manual_Visual_editor_loadmedia.gif]]

==== Insert formulas ====
To insert mathematic formulas in the text, just click the <code>Insert</code> option in the VisualEditor toolbar and select <code>Formula</code>: this will open up the Formula Editor window which will allow you to write math formulas easily.

==== Insert templates ====
Templates are used to add recurring messages to pages in a consistent way, to create navigational boxes, and to provide cross-language portability of texts.

To insert templates (which are needed for <code>Examples, Definitions, Theorems</code>)  you can use the <code>Insert->Template</code> function. To see the supported Templates check out [[TEMPLATES|TEMPLATES.]]
