Mallar används för att lägga till återkommande anteckningar på sidor på ett konsekvent sätt, för att skapa navigeringsrutor, och för att tillhandahålla avancerade funktioner som flyttbarhet av text.

== Infoga mallar ==

Använd funktionen <code>Infoga -> Mall</code> i [[Manual/The Visual Editor|den visuella editorn]] för att infoga mallar.
Skriv in namnet på mallen som ska infogas, och välj den rätta.

[[File:Theorem_insertion_wtl.gif]]

=== Använda mallparametrar ===

Vissa mallar kan ha namngivna parametrar.
Ett sådant fält kan fyllas i genom att klicka på fältnamnet och ange önskat värde.

== Infoga omgivningar ==
I matematik vill man ofta markera saker som teorem, bevis, definition eller någon annan entitet.
Det går att göra genom att klicka på knappen "Lägg till omgivning" i editorn, och välja önskad omgivning.

[[File:Theorem_title_wtl.gif]]

Observera: När en omgivning läggs till visas text som heter "OMGIVNINGSSLUT" eller liknande. Det är bara av praktiska skäll för att veta att det som skrivs är inne i den valda omgivningen, men det syns inte när sidan visas för användaren.
