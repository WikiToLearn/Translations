VisualEditor är huvudverktyget som man kan använda för att skapa och redigera sidor i WikiToLearn.
Det är mycket enkelt att använda VisualEditor.
Om du redan har en viss erfarenhet av någon sorts programvara för textredigering får du inga problem med att förstå hur VisualEditor fungerar.

== Grundläggande användning ==

=== Aktivera VisualEditor ===
När ett nytt avsnitt skapas ska VisualEditor normalt aktiveras: en verktygsrad ska dyka upp längst upp på sidan.

Om en befintlig sida ska redigeras, klicka bara på knappen <code>Redigera</code> längst upp på sidan och se att VisualEditor startar: efter några sekunder ska den vara redo att användas.


=== Redigering ===
VisualEditor har mer eller mindre samma funktioner som alla andra programvaror för ordbehandling, som fetstil, kursivstil, ordnade listor med mera.

=== Avancerad redigering ===

==== Infoga media ====
För att infoga medier i texten, klicka bara på alternativet <code>Infoga</code> i VisualEditors verktygsrad och välj <code>Medier</code>. Det går antingen att ladda upp nya filer genom att använda fliken <code>Ladda upp</code> eller <code>Sök</code> för att söka efter befintliga medier både på WikiToLearn eller Wikimedia Commons.

==== Infoga formler ====
För att infoga matematiska formler i texten, klicka bara på alternativet <code>Infoga</code> i VisualEditors verktygsrad och välj <code>Formel</code>. Det visar fönstret med formeleditorn, som gör det möjligt att enkelt skriva in matematiska formler.

[[File:Math_insertion_wtl.gif]]

==== Infoga mallar ====
Mallar används för att lägga till återkommande meningar på sidor på ett konsekvent sätt.
De används ofta i matematiska avsnitt där man vill skriva en definition, ett teorem, ett lemma, en följdsats, etc.
De kan infogas i vilket avsnitt som helst i kursen.

Funktionen <code>Infoga -> Mall</code> kan användas för att infoga mallar. För att ta reda på mer on mallar och få exempel på de man ska använda, titta på  [[Manual/Using templates|mallsidan]].
